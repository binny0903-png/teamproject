# 레시피허브
레시피허브는 누구나 쉽게 레시피를 공유하고, 요리에 집중할 수 있는 커뮤니티형 플랫폼입니다.  
초보자는 따라하기 쉬운 레시피를, 숙련자는 새로운 도전과 정보 공유를 경험할 수 있습니다.

# 개발 목표
- 요리를 처음 시작하는 사람도 쉽게 접근 가능한 환경 제공
- 숙련자는 새로운 도전과 정보 공유 가능
- 직관적인 레시피 및 질문 게시판 기능 구현

# 사용 기술
- Front-end: HTML, CSS, AJAX
- Back-end: Spring
- Database: Oracle (SqlDeveloper)
- 협업: GitHub

# 전체 페이지
## 사용자
- 로그인 / 회원가입
- 메인페이지
- 레시피 게시판
- 질문 게시판

## 관리자
- 관리자 페이지
- 공지사항 게시판

## 공통
- 헤더 / 푸터

# 담당 역할
- 질문 게시판
  - 목록 페이지
  - 상세 페이지
  - 글쓰기 (작성)
  - 수정
  - 삭제
  - 페이징 처리
