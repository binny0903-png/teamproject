# 레시피허브
레시피허브는 누구나 쉽게 레시피를 공유하고, 요리에만 집중할 수 있는 공간입니다.
요리를 잘 못하는 사람도 쉽게 따라할 수 있도록 돕고, 요리를 잘하는 사람에게는 새로운 도전을 제공하는 커뮤니티형 요리 플랫폼입니다.

# 개발 목표
1. 요리를 처음 시작하는 사람에게도 친숙한 환경 제공
2. 요리에 능숙한 사람에게 새로운 도전과 정보 공유의 장 제공
3. 쉽고 직관적인 레시피 및 질문 게시판 기능 구현

# 사용 기술
1. Front-end : HTML, CSS, AJAX
2. Back-end : Spring
3. DataBase : SqlDeveloper(Oracle)
4. 협업: github

# 전체 페이지 항목
1. 로그인/회원가입
2. 메인페이지
3. 공지사항 게시판
4. 관리자 페이지
5. 레시피 게시판
6. 질문 게시판
7. 헤더/푸터

# 담당 역할
1. 질문 게시판 목록 페이지
2. 질문 게시판 상세 페이지
3. 질문 게시판 작성 페이지 (글쓰기)
4. 질문 게시판 수정 페이지
5. 질문 게시판 삭제
6. 페이징 처리
